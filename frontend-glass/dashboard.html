Mathematical Foundations

Risk Score (Calibrated Ensemble Mean)
r̂ = (1/B) Σᵢ₌₁ᴮ σ(fᵢ(x)) , B = 5
Predictive Uncertainty
û = √[(1/(B-1)) Σᵢ₌₁ᴮ (σ(fᵢ(x)) − r̂)²]
Isolation Forest Novelty
s(x) = 2^(−E[h(x)] / c(n))
Novel if decision_function(x) < −0.08
Dempster-Shafer Combination Rule
m₁₂(A) = [Σ_{B∩C=A} m₁(B)·m₂(C)] / (1 − K)
K = Σ_{B∩C=∅} m₁(B)·m₂(C)
K ≥ 0.25 → HUMAN_ESCALATE (high evidence conflict)
SHAP Explanation (Shapley Values)
f(x) = E[f(X)] + Σⱼ₌₁ᴹ φⱼ(x)
Top-3 |φⱼ| features form the reason code for PEND cases
